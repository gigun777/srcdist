# TreeNavigation — Empty Modal Demo

Цей архів спеціально містить **тільки порожню модалку**.

- Всередині `.modal-body` **немає жодних елементів** (ані дерева, ані кнопок навігації, ані списків).
- Є лише базові кнопки відкриття/закриття модалки та мінімальні стилі.

## Запуск

Відкрий `index.html` у браузері (подвійний клік).
